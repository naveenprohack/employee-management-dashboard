# Employee Management Dashboard (MERN Stack)

A professional, scalable Employee Management Dashboard built with React, Node.js, and MongoDB.

## 🚀 One-Command Local Setup (Recommended)

This project is fully containerized using Docker to eliminate environment configuration friction. 

### Prerequisites
- Ensure you have [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed and running on your machine.

### Instructions
1. Clone this repository to your local machine.
2. Navigate to the root directory of the project in your terminal.
3. Build and launch all services simultaneously by running:
   ```bash
   docker-compose up --build
